A system that checks for if your session has been logged out and logs in again for you.

How to use it -
a.) First open the index.js file and look for - ' CREDENTIALS & CONFIG' and edit your configurations.
b.) Now save that file.
c.)In your terminal run npm i (this will install all the necessary modules)
d.) Edit the bad file if you need it.
e.) Mkae sure everything is in correct folder.
f.) Run the file on a terminal using node.

(I used it for my college's wifi login page, so there may some code related to that. You can always edit out those things.)
